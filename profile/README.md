## 🚀 Sobre Nós

Nós criamos o **Coletivo Inspira** com a missão de trazer arte, cultura, desenvolvimento, diversão e um novo estilo de vida para a região de Bonito-MS. Agora, nossa rede está se expandindo para o universo da tecnologia!

Aqui é o espaço onde centralizamos nossos códigos, desenvolvemos soluções para nossos projetos e hospedamos os portfólios dos membros da nossa comunidade.

---

## 💻 Programa de Portfólios e Subdomínios

Todo membro do Coletivo Inspira tem a oportunidade de ter seu próprio site profissional. Nós utilizamos o GitHub Pages para hospedar os projetos e vinculamos ao nosso domínio oficial, o `inspira.dev.br`. 

Isso significa que você terá um link personalizado, como `seunome.inspira.dev.br`, para divulgar o seu trabalho!

### 🛠️ Como publicar o seu portfólio no Coletivo

* **Passo 1:** Crie um novo repositório público dentro da organização com o seu nome (ex: `felipe-atene`).
* **Passo 2:** Desenvolva a sua página! Você tem liberdade para usar as tecnologias que preferir, como HTML, CSS, JavaScript ou TypeScript.
* **Passo 3:** Ative o recurso **GitHub Pages** nas configurações (*Settings*) do seu repositório.
* **Passo 4:** Mande uma mensagem no grupo geral da nossa Comunidade no WhatsApp solicitando o vínculo do seu subdomínio. A moderação fará o apontamento para o seu site.
